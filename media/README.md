# Audio companion — the 6-30 / Petrushka note

Sonifications of the note's central claims. **MIDI is the primary, editable artifact** (open in any
DAW/notation editor; change instrument, tempo, transpose); the **WAV** files are convenience renders
(self-contained additive sine synthesis, no soundfont needed) so they play without a synthesizer.
All are regenerated by `make_audio.py` (pure Python + numpy).

| File | What you hear |
|------|---------------|
| `petrushka_chord.mid` / `.wav` | C major triad, then F$\sharp$ major triad, then both together — the Petrushka chord, two triads a **tritone** apart ($\{0,1,4,6,7,10\}$ = 6-30). |
| `t6_symmetry.mid` / `.wav` | 6-30 $=\{0,1,3,6,7,9\}$, then its $T_6$ (tritone) transpose — the **same six pitch classes**: the half-turn invariance that makes the stabilizer $\{T_0,T_6\}$. |
| `orbit_12.mid` / `.wav` | the **twelve $T/I$-forms** of 6-30 in sequence — the navigable dual-group orbit (Lewin's GIS). The 8th chord ($I_1$) is the Petrushka chord itself. |

Pitch classes are realized in one octave from C4 (MIDI note $=60+\mathrm{pc}$). Regenerate:
`python make_audio.py`.
